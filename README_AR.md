# منصة اكتشف طرابلس

منصة ترويج سياحي وثقافي لمدينة طرابلس، تركز فقط على:

1. طرابلس القديمة والشواهد التاريخية.
2. الموروث الثقافي والصناعات التقليدية.
3. المذاق الليبي وتجربة الطعام.
4. الخدمات السياحية ومعلومات الزائر.
5. الحياة اليومية والمسارات السياحية.
6. خريطة اكتشاف طرابلس من ملف KML/GeoJSON.

## الملفات المعتمدة في المحتوى
- أشهر الأكلات الليبية.docx
- الصناعات التقليدية.docx
- العملة الوطنية.docx
- طرابلس القديمة.docx
- طرابلس مدينة الألف جولة.docx

## الهدف
تحويل طرابلس إلى تجربة رقمية قابلة للترويج، بحيث يعرف الزائر: أين يذهب، ماذا يرى، ماذا يأكل، ماذا يشتري، وكيف يعيش يومًا سياحيًا في المدينة.

## طريقة التشغيل
افتح ملف `index.html` مباشرة، أو انشر المجلد كاملًا على GitHub Pages أو أي استضافة رسمية.
